Features

Why you'll leave it running

A quiet, native companion for agentic coding — built to be glanced at, not stared at.

✦

Approve from the notch

When your agent wants to run a command or edit a file, the island expands with Always allow / Allow once / Deny. Clicking never steals focus.

▣

Rules that learn

Always allow saves a rule like Bash:npm — matching requests auto-approve from then on. Plain JSON you control.

●

Live session status

A dot pulses while the agent works and turns orange when it needs you. Hover to peek at every session and its last prompt.

⛨

Fails open, fails safe

Relay down? No answer in time? Your agent falls back to its normal terminal prompt. Nothing is ever approved without you.

⌁

Zero-dependency core

A single-file Node relay and a native SwiftUI island built by plain swiftc. No Electron. ~400 KB.

↑

Stays fresh

The island tells you when a new version ships. claude-widget update pulls, rebuilds, and restarts everything.

How it works

Three small parts, no magic

Everything runs locally on your Mac. The only network traffic is localhost, protected by a bearer token.

01

Hooks

Your agent's lifecycle hooks forward permission requests and status events. The gate hook waits for your decision.

02

Relay

A local server holds session state, pending requests, and your always-allow rules — one JSON endpoint any client can speak.

03

Island

The notch app polls every 1.5s and expands the moment something needs you. An experimental iOS widget speaks the same protocol.
